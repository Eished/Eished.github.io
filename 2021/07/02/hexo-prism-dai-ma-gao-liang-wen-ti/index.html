hexo prism 代码高亮问题

hexo-prism-plugin 代码标注只支持小写英文，不支持符号、大写。

prism 支持的语言： https://prismjs.com/#languages-list

  • 小写
#include <iostream>
int main(){
    return 0;
}
  • 大写
#include 
int main(){
    return 0;
}
  • 大写+符号
#include 
int main(){
    return 0;
}
  • 小写+符号
#include 
int main(){
    return 0;
}
